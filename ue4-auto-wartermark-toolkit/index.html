Auto Watermark Toolkit

UE4 Watermark
UE4
Publish Date:   2020-05-24
  • This tool can help us automatically generate dynamic watermark
  • [Unreal Marketplace Link]

Features

  • Dissolve Material Effect
  • Random movement

Steps

  1. Import LOGO: Right-click on the directory and file menu shown below and select Reimport with new file
  2. Import SUPPORT: Right-click on the directory and file menu shown below and selectReimport with new file
  3. Import video: double-click the directory and file shown below to open, and click ... after file path select your video file
  4. Set the video length: double-click the following file
  5. Set the video length: click the button below to modify the end frame number by multiply the number of frames per second by the number of seconds
  6. Sync video length: drag the following clip to sync with the end of the frame
  7. The initial frame number of LOGO: select the frame of LOGO and SURPPORT in the box and drag it to the appropriate starting position
  8. Click Play

Optional

  1. Add custom random range (2) and permanent position (1) : copy the following object and set it in the scene
  2. Configure custom random range and permanent position: select the corresponding object (1), add random range (3) and permanent position configuration (2)
