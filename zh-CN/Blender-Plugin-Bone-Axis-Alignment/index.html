Blender Plugin Bone Axis Alignment

Blender 工具 插件 动画
插件
发布日期:   2022-07-01
  • 这是一个blender插件，可以帮助将骨架转换为 Epic骨架
  • ArtStation 连接
  • BlenderMarket 链接

特性

  1. 在编辑模式下翻转骨骼的轴
  2. 解析骨骼树以生成对齐组
  3. 创建IK骨骼

步骤

轴对齐

  1. 填写 Exclude filter 以排除所有不需要的骨骼
  2. 点击 Parse Armature 按钮
    它会沿着骨骼树的分支形成骨骼组
  3. 点击要选择的骨骼组
  4. 选择轴的目标轴，并通过单击该轴旁边的复选框来翻转它
  5. 点击 Select 显示所选骨骼组的所有骨骼
  6. 根据所需添加或删除骨骼（在目标骨骼上 Shift + 左键），然后点击 Assign
  7. 点击 ApplyAlignment 执行翻转

创建 IK 骨骼 （在确认轴向之后）

  1. 点击 ParseArmature，它会自动填写相应的骨骼名称
  2. 点击 Create IK Bones
