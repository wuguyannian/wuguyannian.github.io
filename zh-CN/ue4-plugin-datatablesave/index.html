表格保存

虚幻4 插件
插件
发布日期:   2020-01-06
  • 虚幻商城 链接
  • 示例项目

这个插件帮助我们程序化地生成游戏关卡数据

  1. 程序化保存图块地图
  2. 程序化保存生成信息
  3. 帮助地图编辑器保存编辑信息

特性

  • 使用字符串查找 表格
  • 清空 表格
  • 保存一行到 表格
  • 移除一行从 表格
  • 保存 表格

步骤

  1. 创建一个 蓝图结构体 并使用它创建一个 表格
  2. 使用 字符串 查找 表格, 或者您可以稍后自己选择该文件。

    最好 右键点击 文件, 并选择 复制引用

  1. 如果需要，清空表格, 使用 Clear DataTable 清空 表格
  2. 使用 RowName 保存一行到表格
  3. 使用 RowName 移除一行从表格
  4. 保存表格的修改
